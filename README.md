# Eoghan

To start an app, run:

`npm run start -w=<name>`

To build an app, run:

`npm run build -w=<name>`

## Publishing Packages

To publish a package to the registry, run:

```sh
npm run start -w=eoghan
npm run build-css-modules -w=@eoghanmccarthy/colors
npm run build -w=@eoghanmccarthy/colors
```
